# mvhost
Npm for create simple virtualhost for apache2.

###Installation
```
sudo npm install mvhost --global
```

###Usage

Enter superuser mode in terminal 
```
sudo -s
```

Create a new virtualhost
```
mvhost --create
```

Remove a existing virtualhost
```
mvhost --remove
```

For help
```
mvhost --help
```
